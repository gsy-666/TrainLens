## 测试 example_cv_train.py

### 独立测试

```bash
python examples/example_cv_train.py \
  --train ./dataset/train \
  --val ./dataset/val \
  --epochs 5 \
  --lr 0.001 \
  --batch 32 \
  --device auto \
  --log test_metrics.jsonl
```

### Dashboard 测试

1. 启动: `start_trainlens.bat`
2. Script Path 改为 `examples/example_cv_train.py`
3. 点击 Start Training
4. 观察实时训练
