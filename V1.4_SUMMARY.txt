## 第一次使用操作

1. 解压 TrainLens_V1.4.zip
2. 确保安装 Python 3.10 或 3.11（勾选 Add to PATH）
3. 双击 setup_trainlens.bat（等待 2-5 分钟）
4. 双击 start_trainlens.bat
5. 浏览器自动打开 http://localhost:8501

## 怎么打包发给别人

### 清理步骤

```batch
rmdir /s /q .venv
rmdir /s /q venv
rmdir /s /q runs\exp_*
del trainlens_config.json
```

可选清理（VS Code 开发文件）：
```batch
rmdir /s /q .vscode
rmdir /s /q node_modules
rmdir /s /q out
rmdir /s /q src
```

### 打包为 ZIP

方式 A - Windows 资源管理器：
- 右键 TrainLens 文件夹
- 发送到 → 压缩文件夹
- 重命名为 TrainLens_V1.4.zip

方式 B - PowerShell：
```powershell
Compress-Archive -Path TrainLens -DestinationPath TrainLens_V1.4.zip
```

### 分发方式

1. 网盘分享（百度网盘、阿里云盘）
2. GitHub Release
3. 内部服务器/邮件

文件大小：约 1-2 MB（不含虚拟环境）
