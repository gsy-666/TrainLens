# TrainLens V1.1 - 实验记录功能

## 新增功能

### 1. 自动实验编号
- 每次点击 Start Training 自动创建 `runs/exp_001`, `runs/exp_002`, `runs/exp_003`...
- 自动扫描已有实验目录，生成下一个编号
- 侧边栏显示 "Next Exp ID"

### 2. 实验配置保存
每次训练开始时保存 `config.json`

### 3. 训练日志保存
- subprocess 的 stdout/stderr 写入 `runs/exp_xxx/train.log`
- 可在 Experiment History 查看最后 50 行日志

### 4. 实验结果归档
训练结束后自动：
- 复制 `runs/current/metrics.jsonl` → `runs/exp_xxx/metrics.jsonl`
- 生成 `summary.json`

### 5. 实验历史界面
新增 "Experiment History" Tab

### 6. 实验详情查看
选择任意实验可查看完整指标和配置

## 修改的文件

### trainlens_app/app.py
完全重写，新增：
- `get_next_exp_id()` - 扫描并生成下一个实验编号
- `create_exp_dir()` - 创建实验目录
- `save_exp_config()` - 保存配置
- `save_exp_summary()` - 保存摘要
- `finalize_experiment()` - 归档实验
- `load_experiments()` - 加载历史
- Tab 布局：Current Training / Experiment History

## 测试步骤

### 1. 启动 Dashboard
```bash
双击 start_trainlens.bat
```

### 2. 运行第一个实验
1. 点击 **Start Training**
2. 侧边栏显示 "Experiment: exp_001"
3. 等待训练完成
4. 训练结束后自动归档

### 3. 检查实验目录
```bash
dir runs\exp_001
```
应该看到：
- `config.json` - 实验配置
- `metrics.jsonl` - 训练指标
- `train.log` - 训练日志
- `summary.json` - 实验摘要

### 4. 查看实验历史
1. 切换到 **Experiment History** Tab
2. 看到实验列表
3. 选择 exp_001 查看详情
4. 展开 "Experiment Config" 和 "Training Log"

## 实验目录结构

```
runs/
├── current/
│   └── metrics.jsonl          # 实时指标
├── exp_001/
│   ├── config.json            # 配置
│   ├── metrics.jsonl          # 指标
│   ├── train.log              # 日志
│   └── summary.json           # 摘要
├── exp_002/
│   └── ...
└── exp_003/
    └── ...
```

## 查看实验记录

### Dashboard UI
1. 切换到 **Experiment History** Tab
2. 表格显示所有实验
3. 选择实验查看详情

### 文件系统
```bash
cat runs\exp_001\config.json
cat runs\exp_001\summary.json
cat runs\exp_001\train.log
```

## 验收标准

- 点击 Start Training 创建 exp_001
- 训练结束后 runs/exp_001 包含 4 个文件
- Experiment History 显示实验列表
- 选择实验可查看详情
- 再次训练创建 exp_002
