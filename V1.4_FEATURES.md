# TrainLens V1.4 - 可分发版本特性文档

## 版本概述

TrainLens V1.4 是一个完全可分发的版本，用户只需解压、安装、启动三步即可使用。

## 新增内容

### 1. 一键安装脚本

**setup_trainlens.bat**
- 使用 `%~dp0` 定位项目根目录，无硬编码路径
- 自动检查 Python 是否可用
- Python 不可用时提供清晰的安装指引
- 创建虚拟环境 `.venv`
- 自动升级 pip
- 安装所有依赖包
- 创建必要目录（runs/current, dataset/train, dataset/val）
- 友好的进度提示和错误处理
- 支持删除旧环境重装

### 2. 一键启动脚本

**start_trainlens.bat**
- 使用 `%~dp0` 定位项目根目录，无硬编码路径
- 检查虚拟环境是否存在
- 不存在时提示先运行 setup_trainlens.bat
- 自动激活虚拟环境
- 启动 Streamlit Dashboard
- 启动失败时保持窗口显示错误信息
- 提供常见问题的解决方案

## 使用流程

### 第一次使用
1. 解压 TrainLens_V1.4.zip
2. 确保安装 Python 3.10 或 3.11
3. 双击 setup_trainlens.bat
4. 双击 start_trainlens.bat

### 以后启动
双击 start_trainlens.bat

## 打包发布

### 打包前清理
```batch
rmdir /s /q .venv
rmdir /s /q runs\exp_*
del trainlens_config.json
```

### 创建 ZIP
```batch
powershell Compress-Archive -Path TrainLens -DestinationPath TrainLens_V1.4.zip
```

总大小：约 1-2 MB（不含虚拟环境）
