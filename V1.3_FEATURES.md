### 1. Dataset Inspector Tab

**trainlens_app/app.py**
- 新增第三个 Tab: "🔍 Dataset Inspector"
- 自动读取侧边栏的 Train Directory 和 Validation Directory
- 支持 ImageFolder 格式数据集检查

### 3. 统计指标展示

**4 个汇总卡片:**
- Train Images: 训练集总图片数
- Val Images: 验证集总图片数
- Classes: 类别总数
- Imbalance Ratio: 类别不平衡比例 (max_count / min_count)

### 4. 类别分布可视化

**类别分布表格:**
- 每个类别的 Train / Val / Total 样本数

**Plotly 分组柱状图:**
- X 轴: 类别名称
- Y 轴: 图片数量
- 蓝色柱: 训练集
- 红色柱: 验证集
- 支持交互式悬停和缩放

### 5. 随机图片预览

**功能:**
- 从训练集随机抽取 8 张图片
- 4x2 网格展示
- 显示每张图片的类别标签
- 使用 PIL Image 加载图片

### 6. 刷新功能

**按钮:**
- "🔄 Refresh Dataset Stats" 按钮
- 重新扫描数据集
- 重新随机抽样图片

### 7. 错误处理

**优雅降级:**
- 路径不存在时显示警告
- 无有效 ImageFolder 结构时提示正确格式
- 图片加载失败时显示错误信息
- 保护性检查防止崩溃

## 使用场景

### 场景 1: 训练前数据集检查

```
用户准备训练模型前:
1. 配置 Train/Val 目录
2. 打开 Dataset Inspector
3. 检查类别数量是否正确
4. 查看类别分布是否均衡
5. 预览图片验证标注
6. 确认无误后开始训练
```

### 场景 2: 数据集质量诊断

```
训练效果不佳时:
1. 打开 Dataset Inspector
2. 检查 Imbalance Ratio (如果 > 5.0 说明严重不平衡)
3. 查看分布图找出样本过少的类别
4. 预览图片检查标注是否正确
5. 根据诊断结果调整数据集或训练策略
```

### 场景 3: 多数据集版本对比

```
切换数据集版本时:
1. 修改侧边栏的数据集路径
2. 点击 Refresh 按钮
3. 对比新旧版本的统计差异
4. 验证数据集更新是否符合预期
```

## 与现有功能集成

- **不影响训练流程**: Dataset Inspector 仅用于检查，不修改数据
- **路径复用**: 自动读取侧边栏配置的路径
